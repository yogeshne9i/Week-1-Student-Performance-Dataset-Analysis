## Project Overview

This project focuses on data cleaning, exploratory data analysis, visualization, and business insight generation using a student performance dataset. The objective is to understand factors affecting student academic performance and demonstrate a professional data analysis workflow.

---

## Dataset Description

The dataset contains information related to student demographics, attendance, study habits, academic scores, and other factors that may influence performance.

### Features Used

* Student ID
* Gender
* Age
* Attendance Percentage
* Study Hours Per Week
* Assignment Score
* Midterm Score
* Final Score
* Parental Education
* Internet Access
* Extra Activities

---

## Libraries Used

* NumPy

* Matplotlib
* Seaborn

## Key Findings

1. Most students scored between 40 and 80 marks.
2. Average final score is approximately 59 marks.
3. Male and female students show similar academic performance.
4. Study hours alone do not strongly predict final scores.
5. Student performance appears to be influenced by multiple factors.
6. Data quality issues were identified and corrected before analysis.

---

## Recommendations

* Provide academic support to low-performing students.
* Encourage consistent study habits and participation.
* Improve data collection quality.
* Monitor multiple performance indicators instead of relying on a single metric.
* Conduct further analysis using larger datasets.
