# Task 1: Student Performance Dataset Analysis

## Objective

Analyze a student performance dataset and perform data cleaning, preprocessing, and exploratory data analysis.

## Dataset Description

The dataset contains student demographic information, attendance records, study habits, and academic performance metrics.

## Activities Performed

* Loaded dataset using Pandas
* Inspected dataset structure
* Identified numerical and categorical features
* Checked missing values
* Removed duplicate records
* Corrected invalid values
* Standardized categorical labels
* Generated descriptive statistics
* Exported cleaned dataset

## Data Cleaning Summary

* Original Records: 206
* Duplicate Records Removed: 6
* Invalid Records Removed: 11
* Final Cleaned Records: 189

## Key Findings

* Average final score is approximately 59 marks.
* Average attendance is approximately 69%.
* Average study hours per week are approximately 10 hours.
* Data quality issues were successfully identified and resolved.

## Tools Used

* Python
* Pandas
* NumPy
